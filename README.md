# QuadraticSolver

Решает квадратное уравнение ax² + bx + c = 0 с коэффициентами произвольной точности. Длинная арифметика написана с нуля на строках — без сторонних библиотек.

## Сборка

```bash
g++ -std=c++17 -Iinc -o QuadraticSolver src/main.cpp src/BigDecimal.cpp src/UnsignedArith.cpp src/QuadSolver.cpp
```

Или через CMake:

```bash
cmake -B build && cmake --build build
./build/QuadraticSolver
```

## Запуск

После сборки программа показывает меню, вводишь коэффициенты и получаешь результат:

```
./QuadraticSolver

1 - Решить уравнение
2 - Выйти
> 1
Введите коэффициенты a, b, c через пробел: 1 -3 2

Уравнение - квадратное
Кол-во корней - 2
x1 = 1
x2 = 2
```

Коэффициенты можно вводить как целые, дробные или в научной нотации — `42`, `-3.14`, `1.5e10`, `1e-3`.

## Тесты

```bash
bash run_tests.sh
```

Прогоняет 23 теста: два корня, один корень, комплексные, линейное уравнение, невалидный ввод, большие числа.

## Структура

```
inc/                 — заголовки (.h)
│   ├── QuadSolver.h
│   ├── BigDecimal.h
│   └── UnsignedArith.h
src/                     — реализация (.cpp)
│   ├── main.cpp             — меню и вывод результата
│   ├── QuadSolver.cpp       — решение уравнения, формула дискриминанта
│   ├── BigDecimal.cpp       — числа произвольной точности
│   └── UnsignedArith.cpp    — сложение, вычитание, умножение, деление строк
tests/                   — тесты
│   ├── test_QuadSolver.cpp
│   ├── test_BigDecimal.cpp
│   └── test_UnsignedArith.cpp
CMakeLists.txt
run_tests.sh
```

Зависимости по слоям: `main` → `QuadSolver` → `BigDecimal` → `UnsignedArith`. Каждый слой знает только о слое ниже — `main` не работает с `BigDecimal` напрямую.

## Как работает

Коэффициенты хранятся как `BigDecimal` — структура с полями `digits` (строка цифр), `exponent10` (степень десятки) и знаком. Дискриминант считается точно через длинную арифметику, квадратный корень — методом Ньютона до 60 знаков.

При D < 0 выводятся комплексные корни в формате `re ± im·i`.
